Airfare Analysis Dashboard
Objectives:-
1.Explore airfare trends across airlines, destinations, number of stops, and time of day
2.Identify top 5 airlines by highest average airfare
3.Understand impact of stops on pricing
4.Find cheapest time of day to travel
5.Identify most affordable airlines for non-stop flights
Process:-
1.Cleaned and transformed data using Power Query Editor
2.Created calculated column using DAX (Time of Day)
3.Developed key measures (Average, Minimum, Maximum Airfare)
4.Built interactive dashboard in Power BI

Key Insights:-
1.Jet Airways has the highest average airfare
2.Non-stop flights are the cheapest; fares increase with more stops
3.Night flights offer the lowest average airfare
4.SpiceJet provides the cheapest non-stop fares. (Min airfare- rs 1759 , Max airfare- rs 23K , Average airfare - rs 4.34K )
5.New Delhi and Cochin have higher fares, while Delhi, Hyderabad, and Kolkata are relatively cheaper
Recommendations
1.Focus more on non-stop routes as they are cost-effective and preferred
2.Promote night flights to increase occupancy
3.Budget airlines like SpiceJet can highlight low-cost non-stop fares as a competitive advantage
